# G-chain

Overview
----------
G-chain is blockchain based fund tracking system.


Features
----------
G-chain features :-
* Transparency in fund transfer
* Tracking of the asset easily
* Immutable Record
* Distributed network
* Decentralised storage of ledger
* Tamper proof
